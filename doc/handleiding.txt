

- Kopieer uit GitHub de map GitHub\threedi-turtle-scripts\sql-scripts\rastercaster naar je projectmap
- Maak een database aan (lokaal) of kies een bestaande database voor de rastercaster
- Run de volgende scripts op deze database: 000_callbackfuncties.sql, 000_functie_leesinstelling, 001_raamwerk en alle scripts in de map GitHub\threedi-turtle-scripts\sql-scripts\scripts-general\geometry_functions
- Instellingen: pixelsize opgeven indien die anders is dan 0.5 x 0.5 m
- Bronvlakjes inladen in je database, bv. src.rastercaster_inputvlakjes (als single polygons)
- Als het nieuwe raster over een bestaand raster heen geplakt moet worden, kopieer de bestaande hoogtekaart rastercaster\raster onder de naam r0
- in script 002 de database gegevens aanpassen


Verbeterpunten:
    - als je veel rastervlakjes hebt die je exporteert, dan duurt dat vet lang met preview_vrt.bat
    - als je een raster met constante waarde opgeeft met param_1 = NULL dan krijg je een raster met waarde 0
    - triggers op old.geom IS DISTINCT FROM new.geom gaan alleen af als bounding box van de geometrie veranderd
    - definition type default 'custom'
